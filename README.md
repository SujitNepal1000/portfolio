# Sujit Portfolio Website

A modern, responsive portfolio website built with React, Vite, and particles.js.

## Features
- Hero section with animated particles background
- Responsive navigation with hamburger menu for tablet/mobile
- Modern/classic color scheme and sans-serif fonts
- Sections: About, Project, Services, Contact

## Getting Started

1. **Install dependencies:**
   ```
   npm install
   ```
2. **Run the development server:**
   ```
   npm run dev
   ```
3. **Build for production:**
   ```
   npm run build
   ```
4. **Preview the production build:**
   ```
    npm run dev
   ```
